## 📖 项目介绍
本项目是**ESP32 官方技术手册的原创白话文翻译与整理**，由小白独立完成。
针对 ESP32 芯片最核心、最底层的硬件原理（如 GPIO 交换矩阵、IO MUX、寄存器配置等），将官方英文/机翻中文的晦涩内容，完整翻译成通俗易懂的大白话，去除冗余废话，保留全部硬核细节，让你不用再对着天书猜含义，直接看懂硬件到底在干什么。

### 已更新内容
- 《通过 GPIO 交换矩阵的外设输入》：详解外设输入路由、纯输入模式配置、硬件实用技巧
- 《IO MUX 和 GPIO 交换矩阵》：完整梳理「物理总闸+智能路由」底层逻辑与稳定配置方案
- 后续将持续更新：UART、I2C、SPI、RMT、PWM 等所有外设的底层手册翻译
