### 📖 项目介绍
这是作者耗时打磨的**工业级嵌入式底层框架**。
核心目标：**永不炸机、代码干净、注释详尽、拿来即用**。

### 核心亮点
-  **全命门防护**：空指针、重复初始化、脏数据、互斥锁、清零操作。
-  **状态机典范**：按键防抖 + 长按识别，工业级标准。
-  **模块化架构**：结构体 + 函数指针，扩展性极强。
-  **中文详细注释**：全中文标注，适合学习与二次开发。
-  **专属水印**：代码内置作者签名，防止被无良商家盗用。

### 🔧 核心功能演示
- 支持 7 键独立控制（播放、音量加减、静音、上一曲、下一曲、电源）。
- 工业级按键状态机，支持短按 / 长按判定。
- FreeRTOS 多任务安全锁，防止资源冲突。
- 结构体清零，杜绝脏数据引起的玄学死机。
