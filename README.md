📖 项目介绍
ESP32-WeaponCore（军火库内核）
是为嵌入式开发者打造的 军工级、高稳定性、生产级底层框架。
从零手写、无依赖、拿来即用、逻辑清晰、中文注释、
专为 不想炸机、不想玄学死机、不想调半天 bug 的人设计。
核心目标：上电就跑、永远不崩、工业级稳定、量产级可靠。
⚡ 核心亮点（军工级标准）
全命门防护：空指针校验、重复初始化保护、脏数据清零、防越界、防崩溃
工业级按键状态机：防抖 + 短按 + 长按，无误触、不掉事件、量产级标准
模块化解耦架构：配置分离、回调驱动、易于扩展、易于维护
FreeRTOS 安全设计：单定时器任务驱动，低 CPU 占用、无冲突
中文详尽注释：每一行逻辑都有说明，新手也能看懂
版权水印保护：代码内置作者签名，防止无良商家盗用
🔧 已实现核心功能（V1.0.2 正式版）
7 路独立按键驱动（播放 / 暂停、音量 +、音量 -、静音、上一曲、下一曲、电源）
完整按键状态机：空闲 → 消抖 → 短按 → 长按
音量安全边界保护（0~100）
静音 / 取消静音自动记忆音量
歌曲序号上下曲防越界
电源键长按安全关机逻辑
GPIO 64 位标准初始化，无溢出风险
全流程空指针保护，杜绝死机
