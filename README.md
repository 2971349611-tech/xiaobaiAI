## 📖 项目介绍
ESP32-WeaponCore（军火库内核）是**从零手写**的嵌入式军工级底层框架，**无依赖、即拿即用**。
核心目标：**上电就跑、永不死机、逻辑清晰、中文注释、量产级可靠**。
专为 **不想炸机、不想玄学死机、不想花时间调 bug** 的嵌入式开发者设计。

---

## ⚡ 核心亮点（军工级标准）
- 🛡️ **全命门防护**：空指针校验、重复初始化保护、脏数据清零、防越界、防崩溃，杜绝“玄学死机”。
- 🧩 **工业级按键状态机**：防抖 + 短按 + 长按，无误触、不掉事件，量产级标准。
- 🧱 **模块化解耦架构**：配置分离、回调驱动，扩展性强、维护性高。
- 🧑‍💻 **FreeRTOS 安全设计**：单定时器任务驱动，低 CPU 占用、无资源冲突。
- 📝 **全中文详尽注释**：每一行逻辑都有说明，新手也能轻松看懂。
- 🔒 **专属版权水印**：代码内置作者签名，防止无良商家盗用。

---

## 🔧 已实现核心功能（V1.0.2 正式版）
- ✅ 7 路独立按键驱动：播放/暂停、音量+、音量-、静音、上一曲、下一曲、电源。
- ✅ 工业级按键状态机：空闲 → 消抖 → 短按 → 长按，精准识别按键事件。
- ✅ 音量安全管理：0~100 边界保护，音量+/- 精准调节。
- ✅ 静音/取消静音：自动记忆备份音量，符合产品级交互。
- ✅ 歌曲序号管理：上/下一曲防越界，安全无异常。
- ✅ 电源键长按安全关机：逻辑严谨，安全关机流程。
- ✅ GPIO 64 位标准初始化：支持 ESP32 全部 IO 引脚，无溢出风险。
- ✅ 全流程空指针保护：杜绝死机、崩溃。
